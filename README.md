# Portfolio — Abubackar Siddiq

A clean, single-page website for viewing my investment portfolio.

**Live:** [https://abubackar-portfolio.netlify.app/](https://abubackar-portfolio.netlify.app/)

## About

This is a personal portfolio tracker — built purely for clear, organized viewing of holdings, allocation, and returns. It is not a pitch deck or investor report.

## Contents

- Portfolio summary (invested, current value, gain/loss, returns)
- Asset allocation (donut chart + legend)
- Holdings table (allocation %, invested, current value, gain/loss %)
- Portfolio metrics (largest holding, best performer, holding count, cash position)

## Tech

- Single self-contained `index.html`
- No external libraries, CSS, or JavaScript
- Semantic, accessible markup
- Print-friendly (exports cleanly to PDF)

## Run locally

Just open `index.html` in a browser — no build step required.

## Deploy

Hosted on [Netlify](https://www.netlify.com/). Any static host works since it's a single HTML file.
